# explore - EXPLORE mode

Diverge before committing. Produce options, commit to none.

## Produce 2-4 directions

Each direction =
- **Lane**: a trend lane (`trend-snapshot.md` / live pulse) or aesthetic family (`aesthetics.md`).
- **Dials**: `DESIGN_VARIANCE / MOTION_INTENSITY / VISUAL_DENSITY`.
- **Palette + type**: one accent, one type pairing, named.
- **One-line rationale**: who it is for, what it signals, the trade-off.
- **Reference imagery** (optional): generate a board via `assets.md` (gpt-image-2) or cite real references.

Make them genuinely different (e.g. minimalist vs high-end vs brutalist), not three tints of one idea.

## Trend-heavy

Pulse current trends first (`trend-research.md`); each direction should reflect a distinct current lane so the user is choosing among real, modern options.

## Hand off

Present directions side by side. Recommend one with a reason. Do not build until the user picks - then enter the CREATE loop with the chosen direction as the Design Read.

Done when: 2-4 genuinely divergent directions presented + one recommendation; nothing built.
